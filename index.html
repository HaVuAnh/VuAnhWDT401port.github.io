About me

My experience from this course

Before learning this subject, I have never learned to code or intend to learn to code by myself because I don't like things that look complicated and confusing to me, in my head I always think that coding is something that is very difficult, especially for graphic design students.

When starting to learn and practice for the first 1 and 2 days, although it is a bit difficult, it is normal when learning something new.

As the 3rd day of learning come, I understood what I was doing and understood what I should do with the assignment without difficulty and having to ask for help or assistance from others. Apparently now I can create any website with a given theme thanks to this course.Even though I've seen the code work many times when learning this subject , but every time I see the code work, it's still give me the same emotion as the first time I see it.
